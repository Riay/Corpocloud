CorpoCloud Notes - Proyecto prototipo
-------------------------------------
Estructura:
- MainActivity: pantalla principal con botones (Agregar / Ver apuntes)
- AddNoteActivity: agregar o editar apunte (usa DatePicker)
- ListNoteActivity: lista de apuntes (ListView) ordenados por fecha descendente
- ViewNoteActivity: ver detalle, editar o eliminar
- DatabaseHelper: maneja SQLite (tabla notes: id, title, description, date)

Instrucciones:
1. Abrir el proyecto en Android Studio.
2. Sync Gradle.
3. Ejecutar en emulador (recomendado API 33).
4. Probar: Agregar apunte -> Ver apuntes -> Abrir nota -> Editar/Eliminar.

Nota: Este proyecto es un prototipo listo para compilar en Android Studio.
